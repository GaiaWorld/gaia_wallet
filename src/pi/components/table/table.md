表格

## 基础功能

- 调用实例

        <components-table-table>{datas:[["2016-05-02","王小虎","上海市普陀区金沙江路 1518 弄"], ...],title:["日期","姓名","地址"],showBoder:true,maxHeight:200 } </components-table-table>

- 参数介绍
    + 显示数据(datas)
        - 总数据为列表格式
        - 每条数据内部也是列表格式
        - 每条数据长度一致
    + 标题(title)---自选
        - 数据为列表格式
        - 数据长度与每条显示数据长度一致
    + 显示边框(showBoder)--默认为false
    + 控制高度(maxHeight)--默认为不控制


- 该组件的目的是根据传入数据，显示不同类型的表格
